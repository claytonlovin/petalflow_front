Gerar novo Token

Os tokens gerados serão usados para autenticação nas suas automações. O token possui acesso permanente e não expira. Você pode revogar o token a qualquer momento. Após gerar o token, copie-o e armazene-o em um local seguro. Não será possível visualizar o token novamente.

Se você perder o token, será necessário gerar um novo.

O token gerado é único e deve ser tratado como uma senha. Não compartilhe o token com ninguém.
